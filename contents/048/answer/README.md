# 048：解答例
* `find()`
	* 概要：部分文字列によるマッチ判定パターンが(どこかの)部分とマッチした場合、`true`。  
      いわゆる、部分一致判定
	* 使用シーン：対象の文中に含まれる、あるワードを検索する際。

* `lookingAt()`
	* 概要：パターンが先頭からマッチした場合、`true`。  
      いわゆる、前方一致判定
	* 使用シーン：文書のヘッダなどに対して検索を行う際。

* `matches()`
	* 概要：パターンが全体とマッチした場合、`true`。  
      いわゆる、完全一致判定
	* 使用シーン：固有名詞のパターンで検索を行う際。
